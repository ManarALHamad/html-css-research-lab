1. In your own words, what kind of things can HTML put on a web page?

ANSWER:

Html can be useful when it comes to organising the main structure of the web page, its responsible for declaring paragraphs, titles and headers.

ANSWER:

3. Look up one HTML tag. Examples: h1, p, a, img.

Which tag did you choose?

ANSWER: h1

What does that tag do?

ANSWER:

First Heading

4. In the HTML below, how many tags are there?

opening tag <p> and closing tag </p>

5.  In the HTML below, which parts are the tags, and which part is the content?

ANSWER:

<h1> & </h1>

6. How does a closing tag differ from an opening tag?

ANSWER:

/ 

7. What is the difference between a tag and an element

ANSWER:

a tag is like the definition of an element for example if its a paragraph <p></p>.
an element is the content inside the tag for example <p>Hello</p>
